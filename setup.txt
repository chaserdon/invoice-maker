Invoice Maker v 1.0
-------------------

INTALLATING:

1. Upload index.html and invoice.php to your web server.  You can rename index.html to invoice.html or some other file name if this file already exist on your server.

RUNNING:

This program needs items.csv, product.csv and order.csv to work.  

1.  Go into your Yahoo Store.
2.  Click the Orders link under process.
3.  For the export button select a range of orders that you want to select
4.  You will be presneted with a CSV file list.  Please download the items.csv, product.csv and order.csv.  These files will have different name, but don't worry about that the program knowns how to name them
5.  Once you have those files saved in a known director, you will then run INDEX.HTML on your web server.  
6.  From here you can upload the three CSV files along with a logo and a message.
